# ECHO+

**ECHO+** 是一款 Android 平台的多功能串口/网络调试助手，专为嵌入式开发调试场景设计。支持 UDP、TCP 客户端、TCP 服务端等多种通信协议，提供数据收发、显示与分析功能。
